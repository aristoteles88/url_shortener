(inner) randomIntBetweenTwoInts(intA, intB) → {int}

Função que retorna um número inteiro aleatório entre dois inteiros positivos fornecidos. Os parâmetros passados serão avaliados e retornará o valor independente de qual é o maior e qual é o menor inteiro. Caso seja fornecido um número negativo ou o zero, este será considerado 1. Caso sejam passados dois números iguais, será retornado um número inteiro entre 1 e o número passado.
Parameters:
Name Type Description
intA int Inteiro positivo. Caso seja passado um número negativo ou nulo, será considerado 1.
intB int Inteiro positivo. Caso seja passado um número negativo ou nulo, será considerado 1.
